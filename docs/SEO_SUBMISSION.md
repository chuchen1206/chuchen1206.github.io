# SEO Submission Guide

## Already configured in code

- sitemap generation via `@astrojs/sitemap`
- robots file at `public/robots.txt`
- canonical URL on each core page
- OpenGraph and Twitter card meta
- JSON-LD schema for `Person` and `ScholarlyArticle`

## Manual submission steps

### Google Search Console

1. Open [Google Search Console](https://search.google.com/search-console/)
2. Add property: `https://chuchen1206.github.io`
3. Verify ownership (recommended: HTML file upload or DNS)
4. Go to Sitemaps
5. Submit: `https://chuchen1206.github.io/sitemap-index.xml`

### Bing Webmaster Tools

1. Open [Bing Webmaster Tools](https://www.bing.com/webmasters/)
2. Add site: `https://chuchen1206.github.io`
3. Verify ownership
4. Submit sitemap: `https://chuchen1206.github.io/sitemap-index.xml`

## Monitoring

- Check index coverage every 2-4 weeks
- Re-submit sitemap when URL structure changes
- Keep profile links in ORCID/Scholar/LinkedIn/GitHub synchronized
